# JOB API Project - NodeJS

#### Email Validation regex

```
^[a-zA-Z0-9.!#$%&'*+/=?^_`{|}~-]+@[a-zA-Z0-9](?:[a-zA-Z0-9-]{0,61}[a-zA-Z0-9])?(?:\.[a-zA-Z0-9](?:[a-zA-Z0-9-]{0,61}[a-zA-Z0-9])?)*$
```

#### Auth routes

```
POST /api/v1/register
POST /api/v1/login
GET /api/v1/auth/:id
PUT /api/v1/auth/:id
DELETE /api/v1/auth/:id
```

#### Job routes

```
GET /api/v1/jobs
POST /api/v1/jobs
GET /api/v1/jobs/:id
PUT /api/v1/jobs/:id
DELETE /api/v1/jobs/:id
```
